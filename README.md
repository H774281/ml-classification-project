# ML Classification Project

## Project Title
Titanic Survival Prediction using Machine Learning

---

## Introduction
This project predicts Titanic passenger survival using machine learning classification algorithms.

The project includes:
- Data preprocessing
- Model training
- Evaluation metrics
- Cross-validation
- Visualization plots

---

## Dataset Description
The Titanic dataset contains passenger information such as:
- Passenger Class
- Gender
- Age
- Fare

Target Variable:
- Survived (0 = No, 1 = Yes)

---

## Algorithms Used
1. Logistic Regression
2. Random Forest Classifier

---

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
- ROC-AUC Score

---

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

```bash
pip install pandas numpy matplotlib seaborn scikit-learn notebook
```

---

## Run the Project

Start Jupyter Notebook:

```bash
jupyter notebook
```

Open:
```text
ML_Classification_Project.ipynb
```

---

## Results
Random Forest achieved better performance compared to Logistic Regression.

Feature importance analysis showed that:
- Sex
- Age
- Fare
- Passenger Class

were important for survival prediction.
